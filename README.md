# HYRESIS : THE LAST STATEMENT

## 실행 방법
- `index.html`을 브라우저로 열면 바로 실행됩니다.

## 구현 단계(체크리스트)
- 1차: 정적 데이터 + 입력/로그 처리 + 화면 출력 완료
- 2차: 로그 다양화, 타이핑 효과, 사운드 등 연출 강화
- 3차: 문서 무결성 시각화 확장, 엔딩 연출 고도화

## 참고
- 기호 입력은 버튼 또는 키보드로 가능합니다.
- 각 Clause는 길이가 고정되어 있으며 입력 길이가 맞지 않으면 재시도를 안내합니다.
